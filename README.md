## Description

This project implements a simulated virtual disk that perform low-level block access, manages file metadata, tracks free disk space using a bitmap, and simulates disk scheduling using the SSTF algorithm.

Stride (every 256 jump)

---

## Workload Generation

The workloads were created as follows:

- Random: provided file (addresses.txt)
- Sequential: generated by listing values from 0 to 65535
- Looping: small set of addresses (100, 200, 300, 400, 500) repeated multiple times
- Stride: generated by stepping through memory in increments of 256

These workloads were designed to simulate different memory access patterns and test locality behavior.
